NetflixGPT is a movie recommendation web application I developed using React, Firebase, and Gemini AI. The app uses an AI-powered recommendation system to suggest movies and TV shows based on user preferences, utilizing data from TMDB (The Movie Database). It features a responsive user interface built with React and styled using Tailwind CSS, ensuring a seamless and engaging experience across devices. I integrated Firebase Authentication for secure user logins and utilized Firebase Firestore for storing user preferences and history. This project enhanced my skills in React, API integration, and working with backend services like Firebase.

Feb 2022-April 2023

Rent Your Requirements(RYR)

Rent Your Requirements (RYR) is a rental service platform I developed as part of my B.Tech project. The application allows users to rent a wide range of items, including vehicles, cameras, phones, coolers, and other household items. Built using HTML, CSS, JavaScript, and managed with a PHPMyAdmin backend, the platform provides a seamless user experience for browsing, renting, and managing rental items. The system allows users to check availability, place rental requests, and manage their rentals efficiently. This project helped me strengthen my skills in full-stack web development and database management.
